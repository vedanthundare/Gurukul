# Gurukul Platform Student Dashboard

A modern student dashboard platform built with React and Node.js.

## Project Structure

- `client/` - React frontend with Tailwind CSS
- `server/` - Node.js backend with Express and MongoDB

## Getting Started

### Frontend Setup
```bash
cd client
npm install
npm run dev
```

### Backend Setup
```bash
cd server
npm install
npm run dev
```

## Tech Stack

### Frontend
- React
- Tailwind CSS
- Vite

### Backend
- Node.js
- Express
- MongoDB
- Mongoose
